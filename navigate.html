How to navigate through an open PGN file

When you open a PGN file a list of moves is loaded into memory, to browse these use the navigation functions. To take back a move press the left arrow key, to go foward a move press the right arrow key.

To take back all the moves in the game press Ctrl + left, and to go foward all the moves in memory press Ctrl + right, all of these functions are available from the Action menu.
