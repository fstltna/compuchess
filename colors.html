The 'Choose colors' dialog


This dialog box allows you to select the color of the squares on the 2D board.

There are two buttons - one changes the color of the light squares and the other changes the color of the dark squares. When you click any of these buttons a standard windows color selection dialog comes up so you can select a color, this color is then previewed in a small square to the left of the button.
