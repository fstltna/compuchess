The 'Move Suggestion' dialog box (Ctrl + H)


The 'Move Suggestion' dialog box in there to give you a hint, it displays a good move for the current situation. If the suggested move has '(book)' written after it this means that it has been taken from an opening library- this is a good way to learn new openings.

The Move Suggestion dialog has three buttons:

Take - accepts the current hint displayed, the computer will automatically make the move for you.

Next - prompts the computer to display another hint if you are not happy with the current one, in some situations there may only be one suggested move available though.

Cancel - closes the dialog box without taking a hint.
