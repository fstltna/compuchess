Introduction to CompuChess

CompuChess is a powerful 3D chess program for Microsoft Windows 98/ME/2000/XP, its unique 3D interface provides the user with a realistic and interactive view of the chessboard. This allows you to play the game and see the chess board as if it were really in front of you!

CompuChess is Packed with exciting features, it can load and save games in the popular 'PGN' format to ensure maximum compatibility with other programs. From the CompuChess website you can download PGN files which contain thousands of games played by masters which you can browse using CompuChess.
