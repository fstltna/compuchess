CompuChess - Online help

The File menu

New Game (Ctrl + N)
Starts a new game and erase the current one.


Open Game (Ctrl + O)
Opens a PGN or an FEN file and loads the moves into memory. More info


Save Game (Ctrl + S)
Saves a PGN or an FEN file. More info


Display PGN info (Ctrl + I)
Displays the information stored in the header of the currently open PGN file. This information inlcludes:

  • The name of the white player.
  • The name of the black player.
  • The site at which the game was played.
  • The date on which the game was played.
  • The round.
  • The result.

Exit
This exits CompuChess and erases any unsaved information.
