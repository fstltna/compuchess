The Toolbar and it's functions

The toolbar in CompuChess provides the user with an easy way of accessing some of the more commonly used functions of the program.

The following is an explaination of all the buttons.


Starts a new game. (Ctrl + N)


Opens a PGN or FEN file. (Ctrl + O)


Saves a PGN or FEN file. (Ctrl + S)


Displays information on a PGN file currently open.


Pauses the chess clock.


Resets the chess clock.


Rotates the 3D board left or right.


Zooms in, or out of the 3D board. ( + / - )


Makes the 3D view fill the entire screen (Ctrl + E)


Disables the 3D view (recommended for low-end systems). (Ctrl + D)


Takes back all moves in memory to the start of the game. (Ctrl + left)


Takes back 1 move. (left)


Goes foward 1 move. (right)


Goes foward all the moves in memory. (Ctrl + right)


Forces the computer to move. (Ctrl + P)


Play as Black with the computer as the White player. (Ctrl + B)


Suggests a good move for you. (Ctrl + H)
