CompuChess - Online help

The 'Configure openings' dialog

The 'Configure openings' dialog allows you to configure the opening strategy which the computer uses. An 'opening' refers to the first few moves each player makes, CompuChess has an opening book which contains thousands of popular openings.

The 'Computer opening style' section allows you to choose the style of opening the computer plays- for example if you chose 'closed' then the computer would play a defensive, non-aggressive game.

The 'Refer to opening book' checkbox toggles whether or not the computer uses the opening book supplied, if it is turned off then CompuChess will make up it's own opening. Turing this feature off can make CompuChess easier to beat as it will not use tried and tested moves in it's opening play.

The 'Learn new openings' checkbox toggles whether or not CompuChess updates the opening book if it finds very good or bad moves.
