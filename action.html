CompuChess - Online help

The Action menu

Play (force move) (Ctrl + P)
Forces the computer player into making a move, if it is the computer's turn then it will stop its current search and give it's best move. If it is your turn then the computer will make a move for you.

Take Back 1 move (left)
Reverts the game back to the last move in memory, this function can be executed as many times as you like until you reach the start of the game. More info


Go foward 1 move (right)
Plays the next move in memory, note that if you are just playing a game and have not taken back any moves then this option will not be available to you. More info


Take back all moves (Ctrl + left)
Takes back all the moves made so far and places you at the beginning of the game, these moves still remain in memory though, and can be replayed using the 'Go foward' option.


Replay all moves (Ctrl + right)
Replays all moves currently in memory and places you at the end or at the current move in the game.

Next game in file
Moves to the next game in the currently open PGN file- the file must contain more than one game for this function to be enabled.


Previous game in file
Moves to the previous game in the currently open PGN file.


Select a game
Opens a dialog box which displays a list of all the games contained within the currently open PGN file, this can only be done if the PGN file contains more than one game. To select a game click on one of the entries in the list and press the 'OK' button.

Suggest a move (Ctrl + H)
Move suggestion dialog box


Play as black (Ctrl + B)
Allows you to play as black and the computer to play as white. Selecting this rotates the 2D and 3D board 180 degrees to give you a better view, this can be performed at any time in the game.


List moves (Ctrl + L)
Displays a dialog box containing all the moves made in the game so far.
