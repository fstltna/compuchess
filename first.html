CompuChess - Online help

Playing my first game

When you start CompuChess you will be playing as white and the computer will be waiting for you to make your move. If you do not know how to move then see 'How do I make a move?'. If you wish to play as Black then either press Ctrl + B or go to: 'Action | Play as Black' on the menu.

Once you have moved CompuChess will start thinking about it's move and then make it, When you first start CompuChess it is set to quite an easy difficulty setting and so it should respond quickly, to change it's playing strength see the 'Computer play options' dialog information. Unless set otherwise, CompuChess will play its first few moves from a book of standard openings, the current opening can be seen in the bottom right of the screen.

Carry on moving until one side wins or the game is drawn, CompuChess will automatically inform you of these events.

To take back a move press the left arrow key, note that this will only take back half a move so to get back to your previous move you have to press left twice. To go foward a move press the right arrow key, again this only goes foward half a move. You will not be able to go foward any moves if you have not take any back first though.

To take back all the moves in the game so far press Ctrl + left, and to go foward all the moves which have been played press Ctrl + right, all of these functions are available from the Action menu.

If you get stuck and need a hint then you can press Ctrl + H or go to: 'Action | Suggest a move', this will bring up the 'Suggest a move' dialog box, for more information on this click here.

Should you want CompuChess to make a move for you then press Ctrl + P or go to:
'Action | Play (force move)' -this will cause the computer to make a move for you using the current search depth.
